# Carter Corzine — Portfolio Website

This repository contains my personal data science portfolio website, built with **Vite + React + Tailwind CSS**.  
The site showcases my data science projects, including work in experimental design, predictive modeling, Six Sigma quality analysis, and machine learning.

**Live site:** [cartercorzine.github.io/Portfolio](https://cartercorzine.github.io/Portfolio)

---

## Features
- Responsive, clean design using **React + Tailwind CSS**  
- Project cards with links to:
  - **GitHub repos** (code)  
  - **PDF reports/demos** (final project results)  
- Easy to update as I complete new projects  

---

## Tech Stack
- [React](https://reactjs.org/) — UI  
- [Vite](https://vitejs.dev/) — build tool  
- [Tailwind CSS](https://tailwindcss.com/) — styling  
- [GitHub Pages](https://pages.github.com/) — deployment
